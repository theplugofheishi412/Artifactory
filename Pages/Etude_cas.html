Contexte

Une équipe développe une application Java et souhaite automatiser son déploiement.

Architecture

  • Git pour le code source
  • Jenkins pour le build
  • Artifactory pour les artefacts
  • Serveur de production

Processus

  1. Push du code sur Git
  2. Build automatique avec Jenkins
  3. Publication du fichier JAR dans Artifactory
  4. Déploiement en production

Résultats

  • Déploiement rapide
  • Moins d’erreurs
  • Traçabilité complète
