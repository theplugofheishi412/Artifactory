Prérequis généraux

  • 4 Go de RAM minimum
  • Docker installé
  • Navigateur web récent
  • Connexion Internet

Installation sur Windows

Étape 1 : Installer Docker Desktop

Télécharger et installer Docker Desktop for Windows. Pendant l’installation, activer WSL 2.

Étape 2 : Vérifier Docker

docker --version

Étape 3 : Lancer Artifactory

docker run -d ^
                --name artifactory ^
                -p 8081:8081 ^
                -p 8082:8082 ^
                docker.bintray.io/jfrog/artifactory-oss

Accès web : 

http://localhost:8081

Installation sur macOS

Étape 1 : Installer Docker Desktop

Installer Docker Desktop pour macOS (Intel ou Apple Silicon).

Étape 2 : Vérifier Docker

docker --version

Étape 3 : Lancer Artifactory

            --name artifactory \
            -p 8081:8081 \
            -p 8082:8082 \
            docker.bintray.io/jfrog/artifactory-oss

http://localhost:8081

Installation sur Linux (Debian)

Étape 1 : Installer Docker

sudo apt update
            sudo apt install docker.io -y
            sudo systemctl start docker
            sudo systemctl enable docker

Étape 2 : Vérifier Docker

docker --version

Étape 3 : Lancer Artifactory

docker run -d \
            --name artifactory \
            -p 8081:8081 \
            -p 8082:8082 \
            docker.bintray.io/jfrog/artifactory-oss

Accès web : 

http://localhost:8081

Connexion à Artifactory

  • Utilisateur : admin
  • Mot de passe : password

Il est recommandé de changer le mot de passe dès la première connexion.

Pourquoi utiliser Docker ?

  • Même installation sur tous les OS
  • Rapide et reproductible
  • Très utilisé en DevOps
  • Facile à intégrer en CI/CD
